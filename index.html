JSON Explicado

O que é JSON?

JSON (JavaScript Object Notation) é um formato de dados leve e de fácil leitura utilizado para troca de informações entre sistemas. Sua estrutura é baseada em texto e é comumente utilizada devido à sua simplicidade e flexibilidade. Este documento explora as vantagens, funcionalidades e uso do JSON.

Vantagens do JSON

  • Simplicidade e Leitura: O JSON é fácil de ler e entender tanto por humanos quanto por máquinas. Sua estrutura é baseada em pares chave-valor, similar à estrutura de um dicionário em Python, o que o torna intuitivo.

Funcionalidades do JSON

  • Estrutura de Dados Simples: Objetos e Arrays.
  • Suporte a Tipos de Dados: Números, strings, booleanos e valores nulos.
  • Facilidade de Uso: Sintaxe simples e legível.
  • Extensibilidade: Aninhar objetos e arrays.

Exemplo de Estrutura JSON

{
  "nome": "João",
  "idade": 30,
  "email": "joao@example.com",
  "interesses": ["Tecnologia", "Viagens"],
  "endereco": {
    "rua": "Rua Principal",
    "cidade": "Cidade A",
    "pais": "Brasil"
  }
}
